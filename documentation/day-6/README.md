 # Day 06 - MongoDB CRUD, Aggregation and Indexing

## Overview

This project demonstrates MongoDB integration with a Node.js backend using Mongoose. It includes schema design, CRUD operations, aggregation pipelines, and indexing for a Portfolio Website.
